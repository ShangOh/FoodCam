# FOODCAM
음식사진을 인식하여 칼로리를 기록하는 앱 (개발기간: 2019.11 ~ 2019.12)

## Demo video (Youtube)
[![IMAGE ALT TEXT](http://img.youtube.com/vi/9TjPmCBvz3E/0.jpg)](https://www.youtube.com/watch?v=9TjPmCBvz3E "[FOODCAM] 포트폴리오 영상")


## Features
- 네이버/카카오/구글 SNS 로그인 구현 
- 자동로그인 및 회원 인증 
- 칼로리/영양소 차트 UI 구현
- Restful API 서버 구축 및 통신
- Yolo V3 모델을 사용하여 음식 이미지 학습 및 인식 - 사용자가 찍은 사진이 음식인지 분류하여 자동저장 

## Technical Stacks
- Nginx, Flask, MariaDB, SQLite
- Tensorflow Lite, CNN
- Retrofit
- Naver login, Kakao login, Google API
